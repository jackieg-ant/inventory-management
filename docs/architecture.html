Data Flow — Filter Pipeline

  1. User selects filters
    FilterBar.vue binds to shared refs in useFilters.js — Time Period, Warehouse, Category, Order Status.
  2. View calls API client
    Page view (e.g. Orders.vue) watches filters and invokes api.getOrders(filters) in api.js.
  3. Axios builds query string
    api.js appends non-'all' values to URLSearchParams and issues GET /api/orders?warehouse=…&status=….
  4. FastAPI filters in memory
    apply_filters() and filter_by_month() reduce the in-memory list with list comprehensions; quarters expand via QUARTER_MAP.
  5. Pydantic validates response
    Result is serialized through response_model=List[Order], guaranteeing schema correctness.
  6. Vue computed properties derive UI state
    Raw response stored in ref(); charts, totals and tables read from computed() for cached reactivity.
